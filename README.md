# SCOUTAI - Android Web View Application

This is a high-performance Android application built as a wrapper for the SCOUT AI web platform.

## Features
- **In-App Browsing:** Seamlessly integrates `https://scout-team40.vercel.app/` using Android WebView.
- **Splash Screen:** Professional logo animation on startup.
- **File Upload Support:** Full support for system file pickers within the web interface.
- **External Redirection:** Handles `mailto:`, `tel:`, and other external app intents.
- **Modern UI:** NoActionBar theme for a full-screen, native-app feel.

## Tech Stack
- **Language:** Kotlin
- **Build System:** Gradle (KTS)
- **UI:** XML Layouts & WebView

## Setup
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run on an Android device or emulator.
